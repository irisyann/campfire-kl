# campfire-kl
Say hi!

1. Create a new branch and name it whatever you like. Example: `iris/my-branch`
2. Create a new `.md` file and name it like this: `<your-name>.md`. Example: `iris-an.md`
3. In the file, write your name and a fun fact about yourself
4. Push the changes and open a pull request.
5. Merge your pull request.
